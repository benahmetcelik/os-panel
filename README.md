## Nasıl Kurulur

Bash script üzerinden otomatik kurulum sağlayın
 ```bash
curl -H "Cache-Control: no-cache" \
     -H "Pragma: no-cache" \
     -fsSL "https://raw.githubusercontent.com/benahmetcelik/os-panel/refs/heads/main/install.sh" | bash 
   ```
